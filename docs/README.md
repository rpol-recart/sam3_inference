# Документация SAM3 Inference Server

Добро пожаловать в документацию для сервера инференса SAM3 (Segment Anything 3). Этот проект представляет собой высокопроизводительный сервер для сегментации изображений и видео с использованием передовой модели SAM3.

## Краткое описание функциональности

### Сегментация изображений
- Поддержка текстовых, точечных и прямоугольных промптов
- Кэширование признаков для ускорения обработки
- Фильтрация по порогу уверенности
- Поддержка батчевой обработки

### Сегментация видео
- Отслеживание объектов на протяжении всего видео
- Поддержка нескольких объектов одновременно
- Пропагация в прямом и обратном направлениях
- WebSocket-стриминг для обработки в реальном времени
- Поддержка многопроцессорной обработки

### Архитектура
- Основан на FastAPI с поддержкой асинхронных операций
- Чистая архитектура с разделением на слои
- Управление сессиями для видеообработки
- Поддержка многопользовательского режима

## Быстрый старт

1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

2. Подготовьте веса модели SAM3

3. Настройте переменные окружения

4. Запустите сервер:
   ```bash
   python -m app.main
   ```

5. Обратитесь к документации для получения подробной информации о доступных API-эндпоинтах
